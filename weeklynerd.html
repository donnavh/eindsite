De boekenzoeker

CSS Accessibility Progressive Enhancement

Visual Interface Design

Voor een opdrachtgever (de bieb) een zogenaamde boekenzoeker ontworpen. Jongeren kunnen met behulp van deze applicatie hun volgende boek uitzoeken. Ontworpen in Adobe XD, begonnen met een lo-fi prototype, om vervolgens te blijven testen en zo stapsgewijs naar de mefi en tot slot de hifi te komen. Het ontwerp heb ik hierbij aangepast en ontworpen voor de bieb en deze huisstijl. In dit project laat ik zien hoe ik een visueel ontwerp kan maken, aangepast op de opdracht. Het ontwerpen van een visual interface naar de bijbehorende regels.

The coffee runner

Business Model Web Payments UX

User Centered Design

Dit project laat zien hoe ik onderzoek kan doen naar een probleem en hier een digitale oplossing voor kan ontwerpen. Hierbij zijn de koffiemachines van de HvA gebruikt en henb ik onderzoek uitgevoerd door middel van o.a. interviews, een user trip en deep hanging outs. Vervolgens is deze data door middel van triangulatie gecontroleerd en is er een persona opgesteld. Door data te verzamelen zijn er toen behoeftes en belemmeringen opgesteld voor de gebruiker. Door middel van een ideaalscenario is er uitgebeeld wat er zou moeten gebeuren. Waarbij de gebruiker graag snel klaar wil zijn, door het onthouden van de voorkeuren is de gebruiker efficienter in het proces. Hier is een goede basis opgesteld voor de UX ervaring en ook het begrijpen hiervan om te kunnen ontwerpen.

Dopper project

Sustainability Market Analysis Campaign Development

Voor dit project hebben we gedurende het semester een Nederlands merk gekozen dat we zouden lanceren op een nieuwe markt met een product. In teamverband hebben we hieraan gewerkt. We hebben de markt geanalyseerd en nagedacht over de haalbaarheid van ons idee. Daarnaast hebben we manieren bedacht om het product te promoten en campagnes te ontwikkelen. Ook hebben we geleerd hoe we een marketingplan voor een bedrijf kunnen opstellen.

Uiteindelijk zijn we uitgekomen op het Dopper-project, waarbij we Dopper laten samenwerken met het Spaanse merk Oysho. De Dopper-bekers zullen te koop zijn in de Oysho-winkels. Deze samenwerking is ontstaan vanuit een focus op duurzaamheid. Dit zes weken durende project heb ik samen met een aantal medestudenten grondig onderzocht.

Web design & development

HTML CSS JavaScript

Toen ik aan deze minor begon, vond ik coderen vooral spannend. Ik had er wel interesse in, maar weinig ervaring. Alles voelde nog erg onbekend en ingewikkeld. Toch heb ik bewust gekozen om het onbekende aan te gaan, omdat ik wilde leren en mezelf uitdagen op een nieuw vlak. In de afgelopen maanden heb ik ontzettend veel geleerd. Doordat ik dagelijks met HTML, CSS, JavaScript en andere tools bezig ben geweest, heb ik een stevige basis opgebouwd. Ik merk dat ik nu veel zekerder ben in het schrijven van code en dat ik technische uitdagingen niet meer uit de weg ga. Dingen die eerst onmogelijk leken, zijn nu beter haalbaar geworden. Naast technische kennis heb ik ook op persoonlijk vlak stappen gezet. Ik durf vaker vragen te stellen, werk gestructureerder, en probeer mijn creatieve ideeën niet meer weg te stoppen, maar juist om te zetten in iets tastbaars. Waar ik eerder vooral op veilig speelde, probeer ik nu ook visueel en technisch meer uit te dagen — bijvoorbeeld in mijn projecten en de manier waarop ik een ontwerp benader. Deze minor heeft me niet alleen geholpen om nieuwe vaardigheden te leren, maar vooral ook om met meer vertrouwen en creativiteit te werken aan digitale producten. Het heeft me een boost gegeven in hoe ik kijk naar design én development.

CSS

CSS Design Prototyping

Voor het vak CSS, moest ik een pagina maken met een control panel erin. Alleen wij mochten slechts css gebruiken. Aanvankelijk leek het een uitdagende taak, maar al snel ontdekte ik de kracht en mogelijkheden van CSS. Met zorgvuldige planning en een creatief oog begon ik met het ontwerpen van het control panel. Het was fascinerend om te zien hoe ik met behulp van CSS grids en flexbox de layout kon vormgeven zonder ook maar een regel JavaScript of HTML te schrijven. Kleurenpaletten werden zorgvuldig gekozen om een visueel aantrekkelijke en functionele interface te creëren. Elke knop en slider werd stijlvol vormgegeven met behulp van pseudo-elementen en overgangen, waardoor het paneel niet alleen functioneel was, maar ook esthetisch plezierig. Het was een waardevolle oefening in minimalistisch ontwerp en probleemoplossing, en het resultaat was een verfijnd, interactief control panel dat volledig draaide op de kracht van CSS. Deze ervaring opende mijn ogen voor de veelzijdigheid van CSS en zijn vermogen om complexe ontwerpen te realiseren met enkel stijlregels.

Stage de Contentsalon

Social Media Webbeheer WordPress

Social media- en webbeheer

Voor mijn korte stage heb ik gewerkt bij de Contentsalon, een agency die zich richt op social media in de richting van de beauty branche. Ik heb hen 10 weken lang ondersteund in de werkzaamheden en enorm veel geleerd. Ik heb veel geleerd over de aanpak van social media en strategieën die je kan gebruiken om een merk te laten groeien. Ook wat voor statistieken interesant zijn om te bekijken hoe een merk kan groeien. Verder deden zij producties voor fotoshoots en events van merken waar ik dan ook van te voren moest uitdenken wat er nodig is op zo'n dag en hoe dit uitgevoerd moet gaan worden. Verder heb ik deze tijd een groot project gehad om hun website een herontwerp te geven. Zij waren hier niet tevreden over maar hadden hier zelf geen tijd voor. Dus ben ik hiermee aan de slag gegaan. Ik ben begonnen met een analyse van de huidige website om vervolgens de plannen te bespreken wat zij zouden willen bereiken. Vervolgens heb ik meerdere prototypes opgesteld om te kunnen tonen hoe het eruit zou komen te zien. Daarna heb ik stap voor stap de website verbeterd vanuit wordpress. Hier heb ik dus ook veel van geleerd en wat hier de mogelijkheden van zijn.
